# CyberFX Synth

CyberFX is a powerful, lightweight parametric sound effect generator designed for games and web applications. It provides a stunning visual interface to create, mutate, and randomize high-quality game audio in real-time.

## Key Features

- **CyberFX Engine**: A high-performance synthesis engine based on the ZzFX DSP architecture, but fully extended with the complete feature set of BFXR.
- **Parametric Control**: Fine-tune harmonics, flanger effects, vibrato, compression, and advanced filter sweeps.
- **Multiple Engines**: Includes integrated support for BFXR and ZzFX engines.
- **Native File Formats**: Save and load sounds in the `.cfx` format.
- **Web-Ready**: Built with modern JavaScript and Vanilla CSS for maximum performance and compatibility.
